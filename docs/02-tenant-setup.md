# Tenant setup

## Goal

This section documents the Microsoft Entra ID lab tenant used for the project.

The goal was to create a clean work/school identity environment that could be used to practice IAM administration tasks, including user provisioning, group-based access, Conditional Access, privileged access, and lifecycle workflows.

## Lab tenant

The lab tenant is named **Northstar Identity Lab**.

This tenant is used only for hands-on IAM practice with fictional users, departments, and security groups.

## Setup summary

The tenant setup included:

- Creating a Microsoft work/school tenant
- Accessing the Microsoft 365 admin center
- Accessing the Microsoft Entra admin center
- Preparing the environment for user, group, MFA, Conditional Access, and PIM configuration

## IAM concepts demonstrated

- Tenant setup
- Cloud identity administration
- Microsoft Entra ID environment preparation
- Separation of lab environment from personal or employer systems
- Evidence-based IAM documentation
